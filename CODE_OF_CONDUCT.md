# Code of Conduct

This project aims to be useful to maintainers who already have too much to do.

Please keep discussions:

- Respectful
- Specific
- Evidence-oriented
- Focused on making maintainer automation safer and easier to evaluate

Harassment, personal attacks, or disclosure of private fixture data are not acceptable. Maintainers may moderate or remove issues, comments, and pull requests that make the project harder to participate in safely.
