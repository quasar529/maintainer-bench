# Roadmap

## v0.1

- Repo-local fixture suites for issue triage and PR reviewability.
- Deterministic grading for decisions, labels, required questions, and required mentions.
- Command runner and optional OpenAI runner.
- GitHub Actions report artifact.
- Real-world-inspired anonymized OSS pattern fixtures.

## v0.2

- Fixture diff reports for prompt, model, and `AGENTS.md` changes.
- Better `suggest-baseline` editing flow.
- More example fixture packs based on anonymized maintainer workflows.
- JSON schema export for editor validation.

## v0.3

- Runner adapters for additional agent CLIs.
- Fixture pack validation command.
- Better report grouping by maintainer workflow category.

## Explicit non-goals

- Public model leaderboard.
- SaaS dashboard.
- Automatic PR comment posting.
- Replacing maintainers with an AI reviewer.
